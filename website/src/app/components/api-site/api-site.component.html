timestamp: The UNIX timestamp of the hour that these stats are from.
count_tweets: The total amount of tweets processed by the bot.
count_retweets: The amount of processed tweets that where ReTweets.
count_tags: The times a tag was found in one of the tweets. Warning: ReTweets are not seached for tags.
count_hashtags: The times a hashtag was found in one of the tweets. Warning: ReTweets are not seached for hashtags.
unique_tags: The ammount of unique tags in the database.
unique_hashtags: The ammount of unique hashtags in the database.

Returns an array containing objects like in /stats.
Please note that index 0 will always be the "Work In Progress" object and might change over the course of the current hour.
Also note that we started recording the statistics history when we rewrote everything during the mirgration from Twitterstats to TwitterDB.

timestamp: The UNIX timestamp of the hour that these stats are from.
tags/hashtags: Sorted array containing the top 100 of the category. Each item has two fields:
- name: The name of the tag/hashtag
- count: The amount of times the item was found during the hour

Returns like in /top
The timestamp must be in UNIX Seconds.
Please note that we started recording the history of the top lists when we rewrote everything during the mirgration from Twitterstats to TwitterDB.

name: The name of the requested tag/hashtag.
